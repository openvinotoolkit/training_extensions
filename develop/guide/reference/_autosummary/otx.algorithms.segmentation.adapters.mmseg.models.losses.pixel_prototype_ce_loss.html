Bases: Module, ABC

Prototype based loss.

Computes cross entropy loss beetwen computeted angles (prototypes and pixel embedings) and target. Includes pixel-prototype contrastive Learning and pixel-prototype distance optimization :param loss_ppc_weight: weight for pixel-prototype contrastive loss. Default: 0.001 :type loss_ppc_weight: float :param loss_ppd_weight: weight for pixel-prototype distance loss. Default: 0.01 :type loss_ppd_weight: float :param ignore_index: index to ignore. Default: 255 :type ignore_index: int :param ignore_mode: ignore mode, used for class incremental learning. Default: False :type ignore_mode: bool

Initializes internal Module state, shared by both nn.Module and ScriptModule.

property loss_name#

Loss Name.

This function must be implemented and will return the name of this loss function. This name will be used to combine different loss items by simple sum operation. In addition, if you want this loss item to be included into the backward graph, loss_ must be the prefix of the name.

Returns:

The name of this loss item.
