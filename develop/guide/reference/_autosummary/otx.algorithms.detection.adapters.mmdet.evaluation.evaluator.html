otx.algorithms.detection.adapters.mmdet.evaluation.evaluator.mask_iou(det: Tuple[ndarray, BitmapMasks], gt_masks: PolygonMasks, iou_thr: float) → ndarray[source]#

Compute the intersection over union between the detected masks and ground truth masks.

Parameters:

  • det (Tuple[np.ndarray, BitmapMasks]) – detected bboxes and masks

  • gt_masks (PolygonMasks) – ground truth masks

  • iou_thr (float) – IoU threshold

Note

It first compute IoU between bounding boxes, then compute IoU between masks if IoU between bounding boxes is greater than 0. Detection mask is resized to detected bounding box size and padded to the same size as ground truth mask in order to compute IoU.

Returns:

iou between detected masks and ground truth masks

Return type:

np.ndarray
