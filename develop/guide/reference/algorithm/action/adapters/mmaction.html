This wrapper is not based on the filesystem, but instead loads the items here directly from the OTX DatasetEntity object. It is adapted from AVADataset of mmaction, but it supports other dataset such as UCF and JHMDB.

Implementation of Fast R-CNN for Action Detection.

Add forward_infer function for inference without pre-proposals

add_detector()#

Add Person Detector for inference.

Action classification backbone + Fast RCNN structure use pre-proposals instead outputs from person detector This saves training and evaluation time. However, this method doesn’t support inference for the dataset without pre-proposals. Therefore, when the model infers to dataset without pre-proposal, person detector should be added to action detector.

forward_infer(imgs, img_metas)#

Forward function for inference without pre-proposal.

patch_pools()#

Patch pooling functions for ONNX export.

AVAFastRCNN’s bbox head has pooling funcitons, which contain dynamic shaping. This funciton changes those pooling functions from dynamic shaping to static shaping.
