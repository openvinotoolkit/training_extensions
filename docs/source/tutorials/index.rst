Tutorials
=========

This section reveals how to use features from OTX CLI and OTX SDK, from base to advanced ones.

For OTX CLI it provides end-to-end solutions from installation to model deployment and demo visualization on certain example for each of the supported tasks:

* Classification
* Object Detection
* Instance Segmentation
* Semantic Segmentation
* Anomaly Classification
